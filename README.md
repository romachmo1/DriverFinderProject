# Driver Finder System

Система для подбора ближайших водителей к заказу с использованием различных алгоритмов поиска.

## Описание

Проект реализует механизм подбора 5 ближайших водителей к заказу на прямоугольной сетке размером N*M.

## Алгоритмы

1. **Brute Force** - простой перебор всех водителей с сортировкой по расстоянию
2. **Priority Queue** - использование приоритетной очереди для эффективного выбора
3. **Spatial Partition** - пространственное разбиение на сетку для ограничения поиска
4. **KD-Tree** - использование k-мерного дерева для эффективного поиска в пространстве
